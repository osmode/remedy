A minimal Peepers client

This is a minimal Peepers client (i.e., a MJPEG over HTTP client). It boils down to an image element in the form

<img src="Peeper's URL"/>

Instructions

This browser and the device running Peepers must be on the same network, e.g., your home wifi.

  1. Start Peepers
  2. Enter Peeper's URL into the text box below
  3. Press the Start button

Tested on Chromium Version 24.0.1312.70 (181759) and Firefox 18.0.2.

If this works you may be interested in going full screen.
